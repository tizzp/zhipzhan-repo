# 中国虚拟租金（OOH）估算的数据收集方案

本文档总结了在中国近期GDP统计改革背景下，如何构建各省虚拟租金（Owner-Occupied Housing, OOH）增加值及其占GDP比重的可行数据方案。所有步骤均以公开数据为基础，可直接用于后续实证研究。

## 1. 全国总量拆分

1. 获取国家统计局公布的全国OOH占GDP比重以及普查后修订的省级GDP数据。
2. 选取能够反映各省住宅资本存量的指标作为权重，如住宅建筑面积、住宅固定资产投资或房地产存量价值。
3. 计算公式：
   - 全国OOH总量 = 全国GDP × 全国OOH占比
   - 省级权重 = 省级住宅指标 / 全国住宅指标总和
   - 省级OOH = 全国OOH总量 × 省级权重
   - 省级OOH占比 = 省级OOH / 省级GDP

## 2. 市场租金等价法

1. **租金数据**：利用贝壳找房、安居客、58同城等平台的公开页面或接口抓取各主要城市的月租金（元/月）和建筑面积（平方米），计算平均租金水平。
2. **住房存量**：收集《统计年鉴》中各省城镇住宅建筑面积或住房存量数据。
3. **估算**：省级OOH = 平均租金（元/平方米·月） × 12 × 省级自有住房面积。
4. **校验**：将得到的省级OOH与全国总量拆分结果进行比较，对异常值进行修正。

## 3. 用户成本法

1. **住宅资产价值**：使用各省房价和住宅存量估算住宅资产价值。
2. **成本参数**：选取贷款利率 `r`、折旧率 `d`、维护率 `m`、房产税费 `t`、预期资本收益率 `g`。
3. **公式**：省级OOH = 住宅资产价值 × (r + d + m + t − g)。
4. **数据源**：利率来自人民银行，折旧及维护率可参考OECD/IMF参数；房产税费取地方财政统计数据。

## 4. 住户微观调查外推

1. 从“住户收支与生活状况调查”或CFPS等微观数据集中提取房产自有情况、估计租金和样本权重。
2. 按省份、城镇/农村、收入组别加权计算自有住房租金，外推至总体。
3. 将微观外推结果与其他方法进行对照。

## 5. 行业口径交叉验证

1. 采集各省房地产业增加值、物业服务业营业额、房地产税费等指标。
2. 与估算的OOH数据进行相关性分析和比对，检查逻辑一致性。

## 6. 国际经验借鉴

1. 参考欧盟统计局、美国BEA、OECD等机构对OOH的核算手册与参数设置。
2. 根据国际经验对中国的参数和模型进行校准，确保估算方法具有可比性。

## 7. 数据管理与 reproducibility

- 所有原始数据、处理脚本和结果应统一存放并版本管理。
- 每一步计算需保留日志和可复现的脚本，便于同行评议。

该方案为后续关于“世界与中国的基础研究：大学、政府与产业力量”主题下的虚拟租金改革研究提供了可直接实施的数据框架。
